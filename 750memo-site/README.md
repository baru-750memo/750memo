# 750万メモ LP

GitHub Pagesに置くための1枚LPです。

## 使い方

1. `index.html` と `assets/hero-suv.png` をGitHubのリポジトリにアップロードします。
2. GitHub Pagesを有効化します。
3. 発行されたURLをInstagramのプロフィールリンクに設定します。

## LINEリンクの差し替え

`index.html` 内の以下の仮URLを、自分のLINEリンクに置き換えてください。

```text
https://line.me/R/ti/p/YOUR_LINE_ID
```

## Instagramプロフィール文の例

```text
借金750万・手取り18万から立て直した記録
稼ぎ方じゃなく、最初にやめたこと
↓ 750万メモは下の青いリンク
```

## リール最後のCTA例

```text
G63は結果であって、始まりじゃない。
当時の俺がやめたことを「750万メモ」にまとめた。
数字を見るのが怖い人だけ、下の青いリンクから見てください。
```

## 注意

このLPは「稼ぎ方」「利益保証」「投資助言」に見えにくい表現へ寄せています。LINE内の文章でも、必勝法・稼げる・誰でも変われる・競艇投資の詳細といった表現は避けるのがおすすめです。
